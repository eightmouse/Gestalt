## Description
Overview sums it up pretty well. It is a simple counter app that runs in the background and can be incremented or decremented with hotkeys.

It was built using Python and Tkinter for the GUI, and it was designed to be as lightweight as possible so it can run on low-end machines without causing performance issues.

I built this around the release of Fire Red / Leaf Green to help people shiny hunting.

It is nothing crazy, just a simple counter but... it works!
